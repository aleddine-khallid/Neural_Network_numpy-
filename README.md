# Neural_Network_numpy-
A modular Deep Learning framework built entirely from scratch using NumPy , to learn mathematical foundation of neural networks .

The goal of this project was to implement a fully functional neural network without using high-level frameworks like PyTorch or Keras for the model logic. It achieves 92.4% accuracy on the MNIST handwritten digit dataset.(60,000 training samples, 10,000 test samples)

-Training time uses 20 Epochs 

-Architecture: 784 Input into 128 Hidden (ReLU activation function) into 10 Output (Softmax activation function)

1- Install dependencies:
```bash
pip install -r requirements.txt
```

2-Run the project:
```bash
python main.py
```
